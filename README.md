This repository contains a classic Snake Game built using the Web Technology Stack (HTML, CSS, and JavaScript). 
The project features a grid-based movement system, score tracking with local storage persistence, and responsive controls for both desktop and mobile devices.

🕹️ FeaturesDynamic Grid Gameplay: 
A $30 \times 30$ grid system implemented via CSS Grid for precise snake movement.
Score Tracking: Real-time score updates and a "High Score" feature that saves your best performance to localStorage.
Responsive Controls:

Desktop: 
Use the Arrow Keys (Up, Down, Left, Right) to change direction.
Mobile/Tablet: Interactive on-screen arrow icons appear for touch-based navigation on smaller screens.
Collision Detection: The game monitors for both wall collisions and self-collisions (hitting the snake's own body).
Adaptive Design: Styled with a dark, modern theme using "Open Sans" and adjusted via media queries for optimal viewing on screens under 800px.


🛠️ Tech StackHTML5: Structure of the game board and control interface.CSS3: Grid layout, Flexbox centering, and mobile responsiveness.JavaScript (ES6):
Game logic, including movement intervals, food randomization, and game state management.Font Awesome: Icons for the mobile control interface.Webhint: Configured via .hintrc for development best practices and accessibility checks.


🚀 Getting Started1. InstallationSimply clone the repository or download the source files.
2. How to PlayOpen index.html in any modern web browser.Use your Arrow Keys to begin moving.
Eat the red food to grow your snake and increase your score.Avoid hitting the walls or your own tail.
3. DevelopmentThe project uses a .hintrc configuration to manage development hints and accessibility rules, such as ignoring specific CSS compatibility issues and ARIA label requirements during the coding phase.
